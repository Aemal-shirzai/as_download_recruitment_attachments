Recruitment Batch Attachments Download

Save time and effort for recruiters or hiring managers who need to review a large number of applications with multiple attachments

Step 1: Install module

To install the module, go to Apps and search for Download Batch Recruitment Attachments name and click on install.


Step 2: Go to recuritment module kanban view.
  • Click of three dots of the announcement.
  • Click on download attachments menu
  • Make sure you the job has applications
  • Click on the Download ZIP File to download the attachments as zip file
  • It will download a zip file with following features:
        Only applicants with attachments will be included.
        Zip file name is Job announcement name with current date.
        Inside zip file there is a folder for every applicant with there name_id_phoneNumber
        Inside that folder all the attachments of the applicant is availible
